# LMS — Система управления обучением

Полноценная веб-платформа для управления обучением с двумя ролями (Администратор и Студент), тёмной темой и адаптивным дизайном.

## Технологии

- **Frontend:** HTML5, CSS3, Vanilla JavaScript (SPA)
- **Backend:** Node.js, Express.js
- **База данных:** MongoDB + Mongoose
- **Авторизация:** JWT
- **Загрузка файлов:** Multer

## Как запустить

### Предварительные требования

1. **Node.js** >= 18
2. **MongoDB** — установленная и запущенная

### Установка

```bash
# 1. Установить зависимости
npm install

# 2. Запустить MongoDB (если не запущена)
# Windows: net start MongoDB
# или запустить mongod вручную

# 3. Запустить сервер
npm start

# или в режиме разработки (с авто-перезагрузкой)
npm run dev
```

### Открыть в браузере

```
http://localhost:3000
```

## Роли

- **ADMIN** — создание курсов, тем, заданий, тестов; управление группами и пользователями; просмотр и оценка работ
- **STUDENT** — просмотр курсов, прохождение тестов, загрузка файлов, просмотр своих оценок

## Особенности

- Тёмная тема в стиле Apple/Tesla
- Mobile-first адаптивный дизайн с бургер-меню
- Плавные анимации и hover-эффекты
- Статусы заданий с цветовой индикацией
- Автоматическая оценка тестов (% правильных ответов × 100)
- Дедлайны с индикаторами времени
- Уведомления в реальном времени
- Прогресс-бары и визуализация активности
